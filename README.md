# Проект: «Куда я поеду»

## Описание проекта

Учебный одностраничный сайт, посвящённый путешествиям.
Проект выполнен по готовому графическому макету с целью отработки навыков
верстки, методологии БЭМ и организации файловой структуры.

Страница содержит несколько логических блоков: шапку сайта, баннер,
секции с описанием мест, блок с отзывами и подвал.

## Функциональность

* Семантическая HTML-разметка
* Адаптивная верстка для экранов разной ширины
* Верстка по методологии БЭМ
* Организация CSS по блокам

## Используемые технологии

* HTML
* CSS
* Flexbox
* Grid Layout
* Методология БЭМ (Nested)
* Адаптивная верстка

## Планы по доработке проекта

* Добавить мобильную версию для экранов менее 768px
* Реализовать переключение языков
* Добавить интерактивность с помощью JavaScript
